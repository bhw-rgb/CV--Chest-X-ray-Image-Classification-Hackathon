# 딥러닝 흉부 이미지 분류 AI 해커톤

[배경]
안녕하세요🙌 오즈코딩 딥러닝 트랙에 참여중인 여러분을 환영합니다!  



본 해커톤은 흉부 X-ray 이미지를 활용한 폐렴 분류 AI 해커톤입니다.  



참가자분들은 제공된 X-ray 이미지를 통해 정상(NORMAL)과 폐렴(PNEUMONIA)을 분류하는 모델을 개발해야 합니다.  



여러분이 지금까지 쌓아온 이론적 지식을 실제 의료 데이터셋에 적용해보며, 실전 문제 해결 능력을 한 단계 성장시켜보세요!  



[주제]  
딥러닝 기반 흉부 X-ray 폐렴 분류 모델 개발  



[설명]
의료 현장에서 촬영된 흉부 X-ray 이미지를 바탕으로 정상/폐렴 여부를 정확하게 분류할 수 있는 AI 모델을 개발합니다.  

- 입력 데이터: 흉부 X-ray 이미지

- 출력: 각 이미지의 정상/폐렴 예측 결과  



[주최 / 주관]
데이콘



[참가 대상]
[초격차] AI 헬스케어 딥러닝 트랙 참여자

1. 평가
평가 산식 : Accuracy score 


2. 참여 규칙 
딥러닝 기반 이미지 분류 모델을 사용하여 학습 및 추론한 결과물을 제출해야 합니다.
1일 최대 제출 횟수 : 10회
학습용 해커톤으로 개인으로 참여하시면 학습 효과가 좋습니다


3. 부정 행위 및 결격 사유
주어진 학습 데이터 이외의 데이터를 학습에 사용
테스트 데이터를 학습에 사용 (* 제출 파일 생성에만 사용해주세요)
외부에 공개된 원본 데이터를 사용


4. 유의 사항
외부 데이터는 사용 불가합니다.
모델 학습에서 검증 혹은 평가 데이터셋 활용 시 Data Leakage로 판단하여 실격 처리됩니다. (부정 행위)
반드시 데이콘에서 받은 데이터만을 사용해주세요.
참가자는 제출 창에서 자신이 최종적으로 채점 받고 싶은 파일을 선택해야 합니다.
